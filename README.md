# CDN

Content Delivery Network Assets

## File browser / downloader

Open `/index.html` to view a simple HTML file tree and download links for CDN assets.
Edit the `cdnFiles` array in `index.html` to list the files you want to expose.

> Note: For cross-origin CDN links, browser download behavior depends on CDN CORS/content-disposition settings. If not configured, files may open in-browser instead of downloading.
